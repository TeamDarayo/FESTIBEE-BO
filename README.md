# DalS2bo Admin

관리자 페이지 - 공연 정보 관리 시스템

## 기능

- 공연 정보 조회
- 공연 정보 등록
- 공연 정보 수정
- 공연 정보 삭제

## 기술 스택

- Next.js 13
- TypeScript
- Tailwind CSS
- Axios

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 환경 변수

`.env.local` 파일에 다음 환경 변수를 설정하세요:

```
NEXT_PUBLIC_API_URL=your_api_url
```
